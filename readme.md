# PWA Base

Template base para criação rápida de **aplicativos PWA profissionais** usando:

- React
- Vite
- TailwindCSS
- Firebase
- API em PHP
- Service Worker customizado
- Manifest completo
- Estrutura preparada para Android (PWABuilder)

Este projeto serve como **base reutilizável** para novos aplicativos.

---

# Tecnologias

Frontend

- React
- Vite
- TailwindCSS
- React Router
- vite-plugin-pwa

Backend

- PHP
- API REST simples
- PDO

Outros

- Firebase
- Service Worker customizado
- Manifest PWA completo
